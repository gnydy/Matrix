# 05 — حدود العمل + ما أحتاجه منك

## ما يمكنني تنفيذه

| البند | التفاصيل |
|-------|----------|
| Monorepo احترافي | Next.js · NestJS · Prisma · Turbo |
| موقع marketing بمستوى Odoo-style | responsive · i18n · dark mode |
| Control Center + ERP modules | حسب الوثائق |
| API كامل + OpenAPI | جاهز للموبايل لاحقًا |
| Docker محلي + قوالب production | على جهازك الآن |
| تكامل دفع | بعد مفاتيح الاختبار منك |
| n8n + ELY | بعد استقرار API |
| توثيق PDF لكل `.md` | `build-all-pdfs.js` |
| CI/CD قوالب | GitHub Actions |

---

## ما لا يمكنني تنفيذه بدونك

| البند | السبب | مطلوب منك |
|-------|--------|-----------|
| **حسابات بوابات الدفع الحية** | تحتاج KYC شركة | تسجيل Stripe/Paymob/… |
| **شهادة SSL production** | تحتاج دومين | شراء دومين + DNS |
| **VPS / استضافة سحابية** | حسابك | IP + SSH أو لوحة |
| **Apple Developer / Google Play** | حساب مطور باسمك | $99 / $25 |
| **محتوى قانوني نهائي** | محامٍ | privacy · terms |
| **شعارات عملاء حقيقية** | إذن العملاء | ملفات PNG/SVG |
| **WhatsApp Business API** | موافقة Meta | Business verification |
| **توقيع عقود عملاء** | خارج النطاق التقني | — |

---

## ما لا أفعله (سياسة)

- لا أضع **مفاتيح حقيقية** في الكود أو Git
- لا أشتري دومينات أو سيرفرات بأموالك
- لا أنشر على App Store باسمي — يحتاج حسابك
- لا أعدّل إعدادات Windows/Git العامة على جهازك بدون طلب
- لا sprint واحد > **10 أيام** — أقسم المهمة
