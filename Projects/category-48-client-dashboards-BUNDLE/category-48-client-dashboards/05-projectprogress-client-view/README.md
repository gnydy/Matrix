# ProjectProgress Client View

Category: 48 - Client Dashboards

## Description
KPIs, reports, services, payments, progress, support, and notifications. This template uses a distinct UI pattern: mobile-first stacked panels with bottom action bar.

## Tech
Next.js, TypeScript, Tailwind CSS, API Routes, Mock Data.

## Install
```powershell
npm install --include=optional --legacy-peer-deps --no-audit --no-fund
```

## Run
```powershell
npm run dev -- -p 3001
```

## API Mock
- `/api/health`
- `/api/records`

## Client Notes
Frontend and backend mock only. Production requires database, authentication, permissions, validation, audit logs, deployment hardening, and real integrations.
