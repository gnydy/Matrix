# RiderFlow Food Delivery

A rider operations template for restaurant pickup, dispatch zones, rider availability, and delivery exceptions.

## Category
Logistics & Delivery Systems — Category 15.

## Target client
Food delivery platform

## Target audience
Food delivery startups, restaurant fleets, cloud kitchens

## Use case
A premium frontend template for logistics, delivery, shipment tracking, route control, fleet monitoring, and dispatch operations. This is a frontend/mock-data template and can later be connected to a real backend.

## Tech stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based structure
- Mock data in `data/site.ts`
- PowerShell scripts for Windows

## Pages
- `/` Overview
- `/tracking` Shipment Tracking
- `/orders` Orders Table
- `/routes` Route Planning
- `/fleet` Fleet & Drivers
- `/dashboard` Operations Dashboard

## How to edit content
Edit `data/site.ts`. It controls brand name, hero text, colors, metrics, shipments, routes, drivers, and modules.

## How to edit colors
Change `accent` and `dark` in `data/site.ts`, then adjust Tailwind classes if needed.

## Delivery notes
This is not a backend delivery platform. It is a sellable UI/UX frontend template prepared for portfolio display, client demos, and future backend integration.
