# PrivateNetwork Portal

Category: 45 - Membership Platforms

## Description
Plans, member areas, premium content, subscriptions, and community access. This template uses a distinct UI pattern: card-based workspace with filter bar and live status rail.

## Tech
Next.js, TypeScript, Tailwind CSS, API Routes, Mock Data.

## Install
```powershell
npm install --include=optional --legacy-peer-deps --no-audit --no-fund
```

## Run
```powershell
npm run dev -- -p 3001
```

## API Mock
- `/api/health`
- `/api/records`

## Client Notes
Frontend and backend mock only. Production requires database, authentication, permissions, validation, audit logs, deployment hardening, and real integrations.
