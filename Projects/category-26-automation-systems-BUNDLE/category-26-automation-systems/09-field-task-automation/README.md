# Field Task Automation

Category: 26 - Automation Systems

## Description
Workflow automation, approvals, triggers, actions, and process orchestration. This template uses a distinct UI pattern: AI assistant workspace with prompt panel and result board.

## Tech
Next.js, TypeScript, Tailwind CSS, API Routes, Mock Data.

## Install
```powershell
npm install --include=optional --legacy-peer-deps --no-audit --no-fund
```

## Run
```powershell
npm run dev -- -p 3001
```

## API Mock
- `/api/health`
- `/api/records`

## Client Notes
Frontend and backend mock only. Production requires database, authentication, permissions, validation, audit logs, deployment hardening, and real integrations.
