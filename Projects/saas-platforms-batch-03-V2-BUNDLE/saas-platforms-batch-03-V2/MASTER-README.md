# SaaS Platforms Batch 03 V2

Permanent rule: templates must not differ by colors only. ## Install all
```powershell
powershell -ExecutionPolicy Bypass -File ".\INSTALL-ALL.ps1"
```

## Run all on different ports
```powershell
powershell -ExecutionPolicy Bypass -File ".\RUN-ALL-20-PORTS.ps1"
```

## Delivery note
These are frontend SaaS templates. They include mock dashboards, pricing, onboarding, mock data, and responsive UI. Production deployment still requires backend, auth, database, billing, role-based permissions, logging, and security hardening.
