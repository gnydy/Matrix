# Permanent Quality Rule for All 1200 Templates

Do not accept any batch where the difference between templates is only colors, text, or images.

Every template must differ in at least five of these areas:

1. UX flow
2. Page composition
3. Navigation pattern
4. Hero structure
5. Business positioning
6. Dashboard/mock interface type
7. Components and controls
8. CTA strategy
9. Information architecture
10. Target client use case

This rule applies to all 60 categories and all 1200 templates.
