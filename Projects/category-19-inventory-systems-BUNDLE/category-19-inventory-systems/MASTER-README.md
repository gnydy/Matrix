# Category 19: Inventory Systems

Products, warehouses, stock movement, low stock alerts, suppliers, and purchase orders.

## Templates
1. **StockPilot Warehouse** — Conversation-first UI with inbox and analytics panels.
2. **LowStock Alert Board** — Left command sidebar, KPI rail, dense admin workspace.
3. **SupplierChain Inventory** — Split-screen hero with form panel and operations cards.
4. **MultiWarehouse HQ** — Board-style operational cards with stages and queues.
5. **BarcodeStock Counter** — Data-heavy table cockpit with filters and status chips.
6. **ColdStorage Inventory** — Executive card grid with insights and feature tiles.
7. **PharmaStock Control** — Timeline and milestone driven workflow.
8. **RetailShelf Replenishment** — Mobile-device preview and app-screen gallery.
9. **AutoParts Inventory** — Desktop shell with window chrome and module launcher.
10. **FoodBatch Traceability** — Prompt/result workspace with usage and model cards.
11. **ConstructionMaterials Yard** — Conversation-first UI with inbox and analytics panels.
12. **FactoryMaterials MRP** — Left command sidebar, KPI rail, dense admin workspace.
13. **EcommerceStock Sync** — Split-screen hero with form panel and operations cards.
14. **HotelSupply Inventory** — Board-style operational cards with stages and queues.
15. **ClinicSupplies Tracker** — Data-heavy table cockpit with filters and status chips.
16. **AssetInventory Register** — Executive card grid with insights and feature tiles.
17. **PurchaseOrder Desk** — Timeline and milestone driven workflow.
18. **CycleCount Mobile Kit** — Mobile-device preview and app-screen gallery.
19. **WarehouseOps Executive** — Desktop shell with window chrome and module launcher.
20. **InventoryOS Enterprise** — Prompt/result workspace with usage and model cards.

## Run
Use `install-one-by-one.ps1`, then `run-all-20.ps1`, then `open-all-chrome.ps1`.
