# StudioSync Creative Booking

## Description
StudioSync Creative Booking is a professional Next.js booking-system frontend template for **Studio & Creative Session Booking**.

## Commercial Use Case
Book studios, equipment, production crews, time blocks, and client requirements.

## Target Client
Photography studios, podcast rooms, content production spaces

## Business Positioning
A creative production booking UI for studios that sell space, equipment, and crew time.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
