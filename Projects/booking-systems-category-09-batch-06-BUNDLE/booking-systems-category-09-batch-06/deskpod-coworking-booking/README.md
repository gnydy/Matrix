# DeskPod Coworking Booking

## Description
DeskPod Coworking Booking is a professional Next.js booking-system frontend template for **Coworking & Meeting Room Booking**.

## Commercial Use Case
Book desks, meeting rooms, private offices, memberships, and visitor access.

## Target Client
Coworking spaces, business centers, shared offices

## Business Positioning
A workspace booking system focused on occupancy, memberships, and room utilization.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
