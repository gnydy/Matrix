# BeautyBar Salon Booking

## Description
BeautyBar Salon Booking is a professional Next.js booking-system frontend template for **Salon Appointment Booking**.

## Commercial Use Case
Book stylists, chairs, treatments, service packages, and repeat customer preferences.

## Target Client
Beauty salons, barbershops, nail studios, grooming lounges

## Business Positioning
A salon booking UI designed for stylist calendars, service bundles, and client loyalty.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
