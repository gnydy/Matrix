# RepairPro Appointment Booking

## Description
RepairPro Appointment Booking is a professional Next.js booking-system frontend template for **Device Repair Booking**.

## Commercial Use Case
Book repair appointments, capture device symptoms, assign technicians, and track repair status.

## Target Client
Phone repair shops, electronics service centers, warranty support desks

## Business Positioning
A repair booking UI that connects customer intake with technician workflow and status tracking.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
