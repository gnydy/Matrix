# CourtReserve Sports Booking

## Description
CourtReserve Sports Booking is a professional Next.js booking-system frontend template for **Sports Court Reservation**.

## Commercial Use Case
Reserve courts, manage memberships, coaches, recurring sessions, and peak-hour pricing.

## Target Client
Padel clubs, gyms, sports arenas, academies

## Business Positioning
A sports reservation system designed around utilization, memberships, and live court availability.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
