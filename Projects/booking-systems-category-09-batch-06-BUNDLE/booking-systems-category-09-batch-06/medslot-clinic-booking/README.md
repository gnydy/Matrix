# MedSlot Clinic Booking

## Description
MedSlot Clinic Booking is a professional Next.js booking-system frontend template for **Medical Appointment Booking**.

## Commercial Use Case
Book patient appointments, assign doctors, monitor waiting list, and manage follow-up visits.

## Target Client
Private clinics, doctor offices, diagnostic centers

## Business Positioning
A clinical booking interface focused on patient flow, doctor availability, and appointment clarity.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
