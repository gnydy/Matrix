# HomeService Dispatch Booking

## Description
HomeService Dispatch Booking is a professional Next.js booking-system frontend template for **Home Services Booking**.

## Commercial Use Case
Book home-service visits, assign technicians, manage addresses, and track arrival windows.

## Target Client
Cleaning, plumbing, maintenance, repair, and field-service companies

## Business Positioning
A dispatch-first booking system for service companies that need operational control.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
