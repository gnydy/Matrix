# Wellness Spa Booking

## Description
Wellness Spa Booking is a professional Next.js booking-system frontend template for **Spa & Wellness Appointment Booking**.

## Commercial Use Case
Book wellness services, therapist schedules, packages, memberships, and room allocation.

## Target Client
Spas, wellness centers, salons, therapy studios

## Business Positioning
A serene booking experience that sells premium wellness packages with operational clarity.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock

This version uses CSS-driven placeholders to keep the template light. Add real images inside `public/` and reference them from components.
