# RentDesk Equipment Booking

## Description
RentDesk Equipment Booking is a professional Next.js booking-system frontend template for **Equipment Rental Booking**.

## Commercial Use Case
Book equipment, check availability, manage deposits, delivery windows, and return schedules.

## Target Client
Equipment rental companies, construction rentals, camera rentals, tool libraries

## Business Positioning
A rental booking interface for companies that need availability and asset-control clarity.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
