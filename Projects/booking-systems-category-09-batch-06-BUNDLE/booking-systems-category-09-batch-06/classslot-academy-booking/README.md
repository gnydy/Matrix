# ClassSlot Academy Booking

## Description
ClassSlot Academy Booking is a professional Next.js booking-system frontend template for **Class & Training Session Booking**.

## Commercial Use Case
Book classes, workshops, instructors, rooms, attendance, and enrollment capacity.

## Target Client
Training centers, academies, workshops, bootcamps

## Business Positioning
An academic booking platform for cohort sessions, workshops, and instructor-led classes.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
