# SuiteKey Hotel Booking

## Description
SuiteKey Hotel Booking is a professional Next.js booking-system frontend template for **Hotel Room Reservation**.

## Commercial Use Case
Sell room nights with availability, guest profiles, booking summaries, and front-desk controls.

## Target Client
Boutique hotels, resorts, serviced apartments

## Business Positioning
A polished hotel reservation UI for properties that need a premium digital booking experience.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
