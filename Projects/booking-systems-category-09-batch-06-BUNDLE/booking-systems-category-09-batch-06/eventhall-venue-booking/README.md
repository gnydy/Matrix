# EventHall Venue Booking

## Description
EventHall Venue Booking is a professional Next.js booking-system frontend template for **Venue & Hall Reservation**.

## Commercial Use Case
Reserve halls, check dates, manage packages, deposits, and event requirements.

## Target Client
Wedding halls, conference venues, event spaces, studios

## Business Positioning
A venue booking system built for capacity, packages, deposits, and event planning clarity.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
