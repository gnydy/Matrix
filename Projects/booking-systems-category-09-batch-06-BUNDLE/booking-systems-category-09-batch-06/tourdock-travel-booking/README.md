# TourDock Travel Booking

## Description
TourDock Travel Booking is a professional Next.js booking-system frontend template for **Travel & Tour Reservation**.

## Commercial Use Case
Book tours, group trips, travel packages, guides, and pickup schedules.

## Target Client
Travel agencies, tour operators, local experience companies

## Business Positioning
A travel booking interface designed for package discovery, itinerary clarity, and group capacity.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
