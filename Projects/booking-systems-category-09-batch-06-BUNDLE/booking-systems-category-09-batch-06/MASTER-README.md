# Category 9 — Booking Systems Batch 06

This batch contains **20 standalone Next.js booking-system templates**.

## Category Definition
Booking systems cover appointments, clinics, hotels, trips, restaurants, consultations, home services, halls, sports courts, salons, repair centers, coworking, and other reservation workflows.

## Batch Quality Target
Each template is a frontend product demo with:
- Booking flow UI
- Availability / calendar UI
- Customer or record table
- Operations dashboard
- Settings mock
- Mock data
- Responsive layout
- Basic SEO metadata
- PowerShell scripts
- Independent ZIP file
