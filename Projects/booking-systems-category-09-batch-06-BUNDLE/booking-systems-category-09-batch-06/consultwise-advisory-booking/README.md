# ConsultWise Advisory Booking

## Description
ConsultWise Advisory Booking is a professional Next.js booking-system frontend template for **Consultation Scheduling**.

## Commercial Use Case
Book advisory calls, qualify leads, assign consultants, and manage consultation packages.

## Target Client
Consultants, advisory firms, coaches, legal/business advisors

## Business Positioning
A professional appointment system for paid advisory sessions and expert consultations.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
