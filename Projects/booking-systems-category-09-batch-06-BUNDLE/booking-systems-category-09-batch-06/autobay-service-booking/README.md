# AutoBay Service Booking

## Description
AutoBay Service Booking is a professional Next.js booking-system frontend template for **Car Service Appointment Booking**.

## Commercial Use Case
Book vehicle service slots, assign bays, record vehicle data, and manage service advisors.

## Target Client
Auto workshops, dealerships, tire shops, car-care centers

## Business Positioning
A workshop booking UI focused on bay capacity, service categories, and advisor workflows.

## Technologies
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based architecture
- Mock data-driven UI
- Responsive layout
- Basic SEO metadata

## Pages
- `/` Overview
- `/booking` Booking flow
- `/calendar` Availability calendar
- `/dashboard` Operations dashboard
- `/customers` Customer/record list
- `/settings` Settings mock
