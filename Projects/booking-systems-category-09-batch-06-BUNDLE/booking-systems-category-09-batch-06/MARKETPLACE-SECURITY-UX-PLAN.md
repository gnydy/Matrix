# Marketplace Security & UX Plan — Booking Systems

Booking templates should be sold as frontend assets with optional custom implementation.

## Marketplace Features
- Category: Booking Systems
- Live demo per template
- Download ZIP per template
- Template detail page
- Case study page
- Request customization CTA
- Compare templates
- License display
- Customer purchase history

## Security Requirements for Final Marketplace
- Authenticated downloads
- Signed expiring download URLs
- Role-based admin dashboard
- Purchase verification before download
- Rate limiting on download endpoints
- Audit logs for downloads and license changes
- File checksum display
- ZIP scanning before publishing
- Admin approval workflow
- No direct public storage URLs for paid templates

## UX Requirements
- Strong search by industry and use case
- Filter by template type, price, complexity, and included pages
- Demo center with category tabs
- Clear "frontend template only" label
- Backend upgrade package CTA
