# MetricLens Analytics Landing

## Commercial Description
Data landing with chart placeholders, report cards, KPI preview, and audit CTA.

## Use Case
Sell dashboards, reports, and data services.

## Target Client
Analytics, reporting, or BI service

## Target Audience
BI consultants, analytics SaaS, agencies

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based structure
- Mock data
- Responsive design
- Basic SEO metadata
- PowerShell scripts for Windows

## Pages / Sections
- Sticky navigation
- Distinct hero section
- Conversion visual / smart UI block
- Feature grid
- Workflow/process section
- Proof section
- Lead capture form
- FAQ
- Footer

## Install
```powershell
npm install
```

## Run
```powershell
npm run dev
```

or:

```powershell
.\run.ps1
```

## Build
```powershell
npm run build
```

or:

```powershell
.\build.ps1
```

## Create ZIP
```powershell
.\zip.ps1
```

## Edit Text
Edit `data/site.ts`.

## Edit Colors
Change `accent` and `bg` inside `data/site.ts`.

## Edit Images
This template uses CSS/HTML visual placeholders. Replace the visual components in `components/LandingPage.tsx` or add images to `public/`.

## Client Delivery Notes
This is a frontend template. It gives the client a strong product/landing experience with mock data. Backend, payments, authentication, CMS, analytics, and API integrations can be added as paid customization.
