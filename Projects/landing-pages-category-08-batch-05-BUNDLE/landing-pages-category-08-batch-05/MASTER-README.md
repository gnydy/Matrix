# Landing Pages — Category 8 Batch

## Scope
This batch contains 20 professional landing page templates. Each template is a standalone Next.js + TypeScript + Tailwind CSS project.

## Commercial Goal
These templates are built for conversion, portfolio display, and future sale inside the MatrixAll template marketplace.

## Category
Landing Pages (20): direct-response pages for products, services, SaaS launches, bookings, events, agencies, fintech, real estate, medical, and digital products.

## Quality Rules
- Every template has a different commercial concept.
- Every template has its own hero message, target audience, and conversion angle.
- Every template uses mock data and reusable components.
- Every template is responsive and has basic SEO metadata.
- Backend is intentionally not included unless requested separately.

## Marketplace Notes
This batch is especially important for the future store because landing pages are fast-selling digital products and can be bundled with customization services.
