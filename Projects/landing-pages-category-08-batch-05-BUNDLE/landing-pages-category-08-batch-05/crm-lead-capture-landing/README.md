# LeadVault CRM Landing

## Commercial Description
Revenue-focused landing page with pipeline preview and lead scoring UI.

## Use Case
Turn visitors into booked sales calls with lead qualification.

## Target Client
CRM provider or sales automation startup

## Target Audience
B2B sales teams, agencies, consultants

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based structure
- Mock data
- Responsive design
- Basic SEO metadata
- PowerShell scripts for Windows

## Pages / Sections
- Sticky navigation
- Distinct hero section
- Conversion visual / smart UI block
- Feature grid
- Workflow/process section
- Proof section
- Lead capture form
- FAQ
- Footer

## Install
```powershell
npm install
```

## Run
```powershell
npm run dev
```

or:

```powershell
.\run.ps1
```

## Build
```powershell
npm run build
```

or:

```powershell
.\build.ps1
```

## Create ZIP
```powershell
.\zip.ps1
```

## Edit Text
Edit `data/site.ts`.

## Edit Colors
Change `accent` and `bg` inside `data/site.ts`.

## Edit Images
This template uses CSS/HTML visual placeholders. Replace the visual components in `components/LandingPage.tsx` or add images to `public/`.

## Client Delivery Notes
This is a frontend template. It gives the client a strong product/landing experience with mock data. Backend, payments, authentication, CMS, analytics, and API integrations can be added as paid customization.
