# Category 18: Accounting Systems

Invoices, expenses, tax, reports, balance sheet UI, and financial dashboards.

## Templates
1. **InvoiceLedger Pro** — Prompt/result workspace with usage and model cards.
2. **ExpenseGuard Finance Desk** — Conversation-first UI with inbox and analytics panels.
3. **TaxBridge Compliance UI** — Left command sidebar, KPI rail, dense admin workspace.
4. **BalanceSheet Executive** — Split-screen hero with form panel and operations cards.
5. **CashFlow Command** — Board-style operational cards with stages and queues.
6. **VendorPay Accounts Payable** — Data-heavy table cockpit with filters and status chips.
7. **ReceivableHub AR Console** — Executive card grid with insights and feature tiles.
8. **SmallBiz Accounting Lite** — Timeline and milestone driven workflow.
9. **RetailFinance POS Books** — Mobile-device preview and app-screen gallery.
10. **ConstructionCost Accounting** — Desktop shell with window chrome and module launcher.
11. **ClinicFinance Billing** — Prompt/result workspace with usage and model cards.
12. **SchoolFees Finance** — Conversation-first UI with inbox and analytics panels.
13. **SubscriptionRevenue Desk** — Left command sidebar, KPI rail, dense admin workspace.
14. **MultiBranch Finance HQ** — Split-screen hero with form panel and operations cards.
15. **AuditTrail Accounting** — Board-style operational cards with stages and queues.
16. **TreasuryControl Board** — Data-heavy table cockpit with filters and status chips.
17. **PayrollAccounting Link** — Executive card grid with insights and feature tiles.
18. **InventoryCost Finance** — Timeline and milestone driven workflow.
19. **RealEstate Finance Desk** — Mobile-device preview and app-screen gallery.
20. **EnterpriseBooks Command** — Desktop shell with window chrome and module launcher.

## Run
Use `install-one-by-one.ps1`, then `run-all-20.ps1`, then `open-all-chrome.ps1`.
