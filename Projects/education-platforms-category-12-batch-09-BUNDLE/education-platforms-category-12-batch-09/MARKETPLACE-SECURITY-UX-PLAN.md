# Marketplace Security & UX Notes — Education Templates

These templates are prepared as digital products for a future template marketplace.

## Marketplace UX
- Each template card should show Live Demo, Download ZIP, View Details, Request Customization, and Compare.
- Add filters: Industry = Education, Complexity, Template Type, Price Range, LMS / Portal / Dashboard.
- Each details page should include screenshots, pages list, target client, license, and customization options.

## Security Plan for the Final Store
- Auth with verified email and 2FA for sellers/admins.
- Signed download URLs with short expiry.
- Purchase ownership checks before ZIP download.
- Audit logs for downloads, license changes, refunds, and admin actions.
- Rate limiting on login, search, and download endpoints.
- Malware scanning for uploaded ZIP files before publishing.
- Separate public demo hosting from private download storage.
- No real student or patient data in demos; mock data only.
