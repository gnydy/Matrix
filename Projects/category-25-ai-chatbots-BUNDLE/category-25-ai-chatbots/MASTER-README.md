# Category 25: AI Chatbots

Customer support bot, sales bot, HR bot, education bot, medical triage mock, and AI agent chat interfaces.

## Templates
1. **SupportBot Helpdesk** — Timeline and milestone driven workflow.
2. **SalesBot Lead Qualifier** — Mobile-device preview and app-screen gallery.
3. **HRBot Employee Assistant** — Desktop shell with window chrome and module launcher.
4. **EduBot Tutor Chat** — Prompt/result workspace with usage and model cards.
5. **ClinicBot Triage Mock** — Conversation-first UI with inbox and analytics panels.
6. **RealEstateBot Advisor** — Left command sidebar, KPI rail, dense admin workspace.
7. **RestaurantBot Ordering** — Split-screen hero with form panel and operations cards.
8. **TravelBot Concierge** — Board-style operational cards with stages and queues.
9. **FinanceBot Expense Coach** — Data-heavy table cockpit with filters and status chips.
10. **LegalBot Intake Assistant** — Executive card grid with insights and feature tiles.
11. **RecruitBot Screening** — Timeline and milestone driven workflow.
12. **CommerceBot Product Guide** — Mobile-device preview and app-screen gallery.
13. **BankingBot Service Desk** — Desktop shell with window chrome and module launcher.
14. **InsuranceBot Claims** — Prompt/result workspace with usage and model cards.
15. **MaintenanceBot Dispatcher** — Conversation-first UI with inbox and analytics panels.
16. **SaaSBot Onboarding** — Left command sidebar, KPI rail, dense admin workspace.
17. **SecurityBot Incident Guide** — Split-screen hero with form panel and operations cards.
18. **AICompanion Chat** — Board-style operational cards with stages and queues.
19. **VoiceBot Control Room** — Data-heavy table cockpit with filters and status chips.
20. **AgentMesh MultiBot Hub** — Executive card grid with insights and feature tiles.
