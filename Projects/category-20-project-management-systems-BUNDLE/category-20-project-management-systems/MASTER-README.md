# Category 20: Project Management Systems

Tasks, Kanban, timeline, teams, milestones, reports, and project dashboards.

## Templates
1. **TaskForge Project Desk** — Left command sidebar, KPI rail, dense admin workspace.
2. **KanbanPilot Studio** — Split-screen hero with form panel and operations cards.
3. **TimelinePro Delivery** — Board-style operational cards with stages and queues.
4. **MilestoneHQ Program Board** — Data-heavy table cockpit with filters and status chips.
5. **AgencyFlow Projects** — Executive card grid with insights and feature tiles.
6. **SoftwareSprint Manager** — Timeline and milestone driven workflow.
7. **ConstructionProject OS** — Mobile-device preview and app-screen gallery.
8. **MarketingCampaign Planner** — Desktop shell with window chrome and module launcher.
9. **ResearchLab Projects** — Prompt/result workspace with usage and model cards.
10. **EventProject Control** — Conversation-first UI with inbox and analytics panels.
11. **LegalCase Project Desk** — Left command sidebar, KPI rail, dense admin workspace.
12. **ManufacturingProject Board** — Split-screen hero with form panel and operations cards.
13. **ConsultingDelivery Hub** — Board-style operational cards with stages and queues.
14. **FreelancerClient Projects** — Data-heavy table cockpit with filters and status chips.
15. **ProductRoadmap Team** — Executive card grid with insights and feature tiles.
16. **DesignApproval Workflow** — Timeline and milestone driven workflow.
17. **OperationsInitiatives HQ** — Mobile-device preview and app-screen gallery.
18. **PortfolioManager PMO** — Desktop shell with window chrome and module launcher.
19. **RiskIssue Tracker** — Prompt/result workspace with usage and model cards.
