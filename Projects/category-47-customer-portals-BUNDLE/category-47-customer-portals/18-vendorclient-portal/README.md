# VendorClient Portal

Category: 47 - Customer Portals

## Description
Tickets, invoices, files, orders, project status, account settings. This template uses a distinct UI pattern: mobile-first stacked panels with bottom action bar.

## Tech
Next.js, TypeScript, Tailwind CSS, API Routes, Mock Data.

## Install
```powershell
npm install --include=optional --legacy-peer-deps --no-audit --no-fund
```

## Run
```powershell
npm run dev -- -p 3001
```

## API Mock
- `/api/health`
- `/api/records`

## Client Notes
Frontend and backend mock only. Production requires database, authentication, permissions, validation, audit logs, deployment hardening, and real integrations.
