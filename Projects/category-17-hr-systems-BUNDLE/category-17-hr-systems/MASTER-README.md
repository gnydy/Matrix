# Category 17: HR Systems

Human resources, attendance, payroll, recruitment, and performance operations.

## Templates
1. **PeopleCore HR Command** — Desktop shell with window chrome and module launcher.
2. **ShiftPulse Attendance Suite** — Prompt/result workspace with usage and model cards.
3. **PayrollBridge Finance HR** — Conversation-first UI with inbox and analytics panels.
4. **TalentGate Recruitment OS** — Left command sidebar, KPI rail, dense admin workspace.
5. **Performance360 Review Hub** — Split-screen hero with form panel and operations cards.
6. **LeaveFlow Employee Portal** — Board-style operational cards with stages and queues.
7. **Onboardly New Hire Journey** — Data-heavy table cockpit with filters and status chips.
8. **HR Legal Docs Vault** — Executive card grid with insights and feature tiles.
9. **CulturePulse Engagement** — Timeline and milestone driven workflow.
10. **TrainingMatrix HR LMS** — Mobile-device preview and app-screen gallery.
11. **BranchHR Multi Location** — Desktop shell with window chrome and module launcher.
12. **ContractorPeople Portal** — Prompt/result workspace with usage and model cards.
13. **ExecutiveHR Analytics** — Conversation-first UI with inbox and analytics panels.
14. **BlueCollar Workforce OS** — Left command sidebar, KPI rail, dense admin workspace.
15. **RemoteTeam HR Desk** — Split-screen hero with form panel and operations cards.
16. **MedicalStaff HR Board** — Board-style operational cards with stages and queues.
17. **FactoryShift HR Console** — Data-heavy table cockpit with filters and status chips.
18. **Recruiter Pipeline Pro** — Executive card grid with insights and feature tiles.
19. **Compensation Planner** — Timeline and milestone driven workflow.
20. **PeopleOps Enterprise HQ** — Mobile-device preview and app-screen gallery.

## Run
Use `install-one-by-one.ps1`, then `run-all-20.ps1`, then `open-all-chrome.ps1`.
