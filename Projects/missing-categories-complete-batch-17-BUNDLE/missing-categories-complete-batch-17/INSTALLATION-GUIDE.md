# Installation Guide

Do not install every category at the same time. Pick one category and install one by one.

```powershell
cd "E:\Matrix\Projects\missing-categories-complete-batch-17\category-03-erp-systems"
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\install-one-by-one.ps1
.un-all-20.ps1
.\open-all-chrome.ps1
```

If SWC errors appear, use Node.js LTS 64-bit and reinstall one template first.
