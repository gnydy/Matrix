# Grocery POS / Cashier Systems — 20 Template Batch

This batch adds 20 grocery cashier POS frontend templates, ranging from a tiny baqala cashier to an enterprise grocery chain operating system.

These templates are frontend demos only. They are built to sell, preview, and customize. They are not production accounting/payment systems until backend, database, roles, taxes, fiscal receipt rules, backups, and security are implemented.

## Run one template

```powershell
cd "path-to-template"
npm install
npm run dev
```

Use `run-all-20.ps1` from this folder.
