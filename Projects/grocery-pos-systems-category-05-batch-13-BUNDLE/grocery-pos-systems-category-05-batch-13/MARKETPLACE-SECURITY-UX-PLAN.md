# Marketplace Security & UX Notes for POS Templates

When these templates are sold in the future marketplace, protect downloads with authenticated purchases, signed URLs, license records, and audit logs.

For real POS production projects, add:
- role-based access control
- cashier permissions
- safe refunds and supervisor override
- immutable receipt numbers
- tax settings per country
- encrypted backups
- branch isolation
- offline sync conflict handling
- audit trails
- payment gateway PCI-sensitive separation
