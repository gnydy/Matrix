# Butcher & Deli Counter POS

Professional Next.js grocery cashier / POS frontend template.

## Commercial description
واجهة بيع للأقسام الطازجة: لحوم، أجبان، مقطعات، وخدمة انتظار.

## Use case
Fresh counter — جزار / ديلي داخل سوبرماركت

## Tech
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based frontend
- Mock data only

## Pages
- `/` Cashier terminal
- `/products` Product catalog
- `/sales` Sales and receipts
- `/inventory` Inventory alerts
- `/shifts` Cashier shifts
- `/reports` Sales reports
- `/settings` Settings mock

## Install
```powershell
npm install
```

## Run
```powershell
npm run dev
```

## Build
```powershell
npm run build
```

## ZIP
```powershell
.\zip.ps1
```

## Edit content
Change `data/site.ts` to update store name, metrics, products, cart, alerts, and positioning.

## Delivery note
This is a frontend sales/demo template, not a production accounting or payment system. Real deployment needs backend, database, user roles, audit logs, receipt numbers, tax rules, backup strategy, and payment integration.
