# BudgetWise Personal Finance

Category: 32 - Fintech Platforms

## Description
Wallets, payments, banking UI, investment dashboards, and financial tools. This template uses a distinct UI pattern: desktop app shell with toolbar, tabs, and resizable panels.

## Tech
Next.js, TypeScript, Tailwind CSS, API Routes, Mock Data.

## Install
```powershell
npm install --include=optional --legacy-peer-deps --no-audit --no-fund
```

## Run
```powershell
npm run dev -- -p 3001
```

## API Mock
- `/api/health`
- `/api/records`

## Client Notes
Frontend and backend mock only. Production requires database, authentication, permissions, validation, audit logs, deployment hardening, and real integrations.
