# Executive Sales War Room

## Commercial Description
Executive overview with pipeline risk, forecasts, strategic accounts, and team performance

## Use Case
A sellable CRM frontend template for Sales directors and founders tracking company-wide commercial performance.

## Target Client
Sales directors and founders tracking company-wide commercial performance

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock CRM data
- Component-based architecture
- Responsive UI

## Included Pages
- Home CRM presentation
- Dashboard
- Accounts
- Pipeline

## Included CRM UI
- Sidebar / top navigation
- Stats cards
- Pipeline board
- Account cards
- CRM table
- Activity timeline
- AI next-action assistant
- Empty-ready structures through mock states
- Role-based UI concept

## Run

```powershell
cd "executive-sales-war-room"
npm install
npm run dev -- --port 3099
```

Open:

```text
http://localhost:3099
```

## Build

```powershell
npm run build
```

## ZIP

```powershell
.\zip.ps1
```

## Customization
Edit:
- `data/crm.ts` for text, deals, accounts, stats
- `components/CRMTemplate.tsx` for UI sections
- `styles/globals.css` for global CSS
- Tailwind utility classes for spacing, layout, and visual identity

## Delivery Notes
This is a frontend CRM template. It is designed for portfolio display, client demos, and future backend integration. It is not a production backend CRM.
