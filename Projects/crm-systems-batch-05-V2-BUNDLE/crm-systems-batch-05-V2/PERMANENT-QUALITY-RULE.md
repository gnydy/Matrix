# Permanent Quality Rule

This rule applies to all future batches in the 60 category × 20 template plan.

A template is rejected if the difference is only:
- color
- copy
- logo/name
- background image

A template is acceptable only when it differs in:
- page structure
- UX model
- navigation behavior
- visual composition
- business positioning
- target client
- core components
- demo use case
- primary CTA
- presentation logic

For CRM systems specifically, each template must include CRM-specific UI such as leads, accounts, pipeline, customer profiles, follow-ups, notes, activity, reporting, and AI/smart assistant where appropriate.
