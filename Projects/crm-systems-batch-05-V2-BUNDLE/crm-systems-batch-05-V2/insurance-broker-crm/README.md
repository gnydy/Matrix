# Insurance Broker CRM

## Commercial Description
Policy lifecycle CRM with renewal alerts, client risk profile, claims, and coverage gaps

## Use Case
A sellable CRM frontend template for Insurance brokers handling renewals, claims, and policy advisory.

## Target Client
Insurance brokers handling renewals, claims, and policy advisory

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock CRM data
- Component-based architecture
- Responsive UI

## Included Pages
- Home CRM presentation
- Dashboard
- Accounts
- Pipeline

## Included CRM UI
- Sidebar / top navigation
- Stats cards
- Pipeline board
- Account cards
- CRM table
- Activity timeline
- AI next-action assistant
- Empty-ready structures through mock states
- Role-based UI concept

## Run

```powershell
cd "insurance-broker-crm"
npm install
npm run dev -- --port 3097
```

Open:

```text
http://localhost:3097
```

## Build

```powershell
npm run build
```

## ZIP

```powershell
.\zip.ps1
```

## Customization
Edit:
- `data/crm.ts` for text, deals, accounts, stats
- `components/CRMTemplate.tsx` for UI sections
- `styles/globals.css` for global CSS
- Tailwind utility classes for spacing, layout, and visual identity

## Delivery Notes
This is a frontend CRM template. It is designed for portfolio display, client demos, and future backend integration. It is not a production backend CRM.
