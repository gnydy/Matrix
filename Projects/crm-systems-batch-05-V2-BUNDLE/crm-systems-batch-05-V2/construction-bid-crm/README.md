# Construction Bid CRM

## Commercial Description
Bid pursuit CRM with tender stages, project value, decision makers, and submission checklist

## Use Case
A sellable CRM frontend template for Construction companies managing tenders, bids, and client pursuit.

## Target Client
Construction companies managing tenders, bids, and client pursuit

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock CRM data
- Component-based architecture
- Responsive UI

## Included Pages
- Home CRM presentation
- Dashboard
- Accounts
- Pipeline

## Included CRM UI
- Sidebar / top navigation
- Stats cards
- Pipeline board
- Account cards
- CRM table
- Activity timeline
- AI next-action assistant
- Empty-ready structures through mock states
- Role-based UI concept

## Run

```powershell
cd "construction-bid-crm"
npm install
npm run dev -- --port 3093
```

Open:

```text
http://localhost:3093
```

## Build

```powershell
npm run build
```

## ZIP

```powershell
.\zip.ps1
```

## Customization
Edit:
- `data/crm.ts` for text, deals, accounts, stats
- `components/CRMTemplate.tsx` for UI sections
- `styles/globals.css` for global CSS
- Tailwind utility classes for spacing, layout, and visual identity

## Delivery Notes
This is a frontend CRM template. It is designed for portfolio display, client demos, and future backend integration. It is not a production backend CRM.
