# E-commerce Retention CRM

## Commercial Description
Retention CRM with customer segments, cart recovery, lifecycle campaigns, and LTV cards

## Use Case
A sellable CRM frontend template for E-commerce brands managing repeat purchase, segments, and abandoned carts.

## Target Client
E-commerce brands managing repeat purchase, segments, and abandoned carts

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock CRM data
- Component-based architecture
- Responsive UI

## Included Pages
- Home CRM presentation
- Dashboard
- Accounts
- Pipeline

## Included CRM UI
- Sidebar / top navigation
- Stats cards
- Pipeline board
- Account cards
- CRM table
- Activity timeline
- AI next-action assistant
- Empty-ready structures through mock states
- Role-based UI concept

## Run

```powershell
cd "ecommerce-retention-crm"
npm install
npm run dev -- --port 3094
```

Open:

```text
http://localhost:3094
```

## Build

```powershell
npm run build
```

## ZIP

```powershell
.\zip.ps1
```

## Customization
Edit:
- `data/crm.ts` for text, deals, accounts, stats
- `components/CRMTemplate.tsx` for UI sections
- `styles/globals.css` for global CSS
- Tailwind utility classes for spacing, layout, and visual identity

## Delivery Notes
This is a frontend CRM template. It is designed for portfolio display, client demos, and future backend integration. It is not a production backend CRM.
