# Real Estate Leads CRM

## Commercial Description
Lead-to-viewing CRM with property matches, viewing calendar, and broker activity

## Use Case
A sellable CRM frontend template for Real estate agencies handling buyers, sellers, viewings, and brokers.

## Target Client
Real estate agencies handling buyers, sellers, viewings, and brokers

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock CRM data
- Component-based architecture
- Responsive UI

## Included Pages
- Home CRM presentation
- Dashboard
- Accounts
- Pipeline

## Included CRM UI
- Sidebar / top navigation
- Stats cards
- Pipeline board
- Account cards
- CRM table
- Activity timeline
- AI next-action assistant
- Empty-ready structures through mock states
- Role-based UI concept

## Run

```powershell
cd "real-estate-leads-crm"
npm install
npm run dev -- --port 3083
```

Open:

```text
http://localhost:3083
```

## Build

```powershell
npm run build
```

## ZIP

```powershell
.\zip.ps1
```

## Customization
Edit:
- `data/crm.ts` for text, deals, accounts, stats
- `components/CRMTemplate.tsx` for UI sections
- `styles/globals.css` for global CSS
- Tailwind utility classes for spacing, layout, and visual identity

## Delivery Notes
This is a frontend CRM template. It is designed for portfolio display, client demos, and future backend integration. It is not a production backend CRM.
