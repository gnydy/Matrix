# Marketplace Notes — Dashboard & Admin Panels

## Product Listing Requirements
Each dashboard template should be listed with:
- Screenshot gallery
- Live demo link
- Download ZIP button
- Template complexity
- Industry
- Included pages
- License terms
- Customization CTA

## Security When Selling Downloads
For the final marketplace, do not expose static public ZIP links for paid templates. Use:
- Authenticated customer accounts
- Purchase records
- Signed temporary download URLs
- Download limits
- License key per purchase
- Admin audit logs
- Anti-abuse rate limiting
- Payment webhook verification
- Separate preview server from download storage

## UX Notes
Dashboard templates need strong preview pages because clients often buy based on perceived system depth. Show sidebar, tables, stats, charts, settings and role concepts in the marketplace screenshots.
