# Permanent Quality Rule

For all future batches in the 60 categories / 1200 templates plan:

Changing colors, names, icons or copy is not enough.
Each template must differ in at least five of these areas:

1. Layout composition
2. Navigation model
3. Hero or command area
4. Data presentation style
5. Target business positioning
6. User flow
7. Component hierarchy
8. Dashboard / portal / website type
9. CTA strategy
10. UX states and interactions
