# ERP Systems Batch 04 V2

This batch contains 20 different ERP frontend templates. Each one uses a different business concept and UX structure, not only color changes.

## Templates

1. **Finance Command ERP** — CFO command center with ledger, cashflow, tax, approvals and executive risk signals — Port `3060`
2. **Factory Floor ERP** — Production floor control with machines, work orders, materials and quality gates — Port `3061`
3. **Inventory Control ERP** — Inventory movement cockpit with bins, stock alerts, suppliers and transfer requests — Port `3062`
4. **Procurement Hub ERP** — Purchase request to vendor comparison workflow with RFQ board and approval lanes — Port `3063`
5. **People Payroll ERP** — Employee records, attendance, payroll cycle, leaves and performance control desk — Port `3064`
6. **Sales Order ERP** — Sales orders, quotations, invoices, client accounts and margin visibility — Port `3065`
7. **Construct Cost ERP** — Project costs, BOQ, site progress, equipment and subcontractor dashboard — Port `3066`
8. **Fleet Route ERP** — Fleet dispatch, route planning, shipments, driver status and delivery SLA control — Port `3067`
9. **Hospital Ops ERP** — Patient flow, departments, billing, stock, claims and resource allocation — Port `3068`
10. **Campus Admin ERP** — Admissions, classes, student fees, teachers, attendance and academic operations — Port `3069`
11. **Property Asset ERP** — Property portfolio, tenants, maintenance, leases and occupancy analytics — Port `3070`
12. **Hotel Chain ERP** — Rooms, bookings, housekeeping, guest accounts and revenue operations — Port `3071`
13. **Retail Branch ERP** — Branches, POS summaries, products, stock transfers and cashier performance — Port `3072`
14. **Restaurant Ops ERP** — Kitchen orders, ingredients, purchases, table revenue and delivery operations — Port `3073`
15. **Service Maintenance ERP** — Work orders, technicians, spare parts, contracts and SLA escalation — Port `3074`
16. **AgriFlow ERP** — Crops, irrigation, harvest batches, sensors, equipment and farm inventory — Port `3075`
17. **Project Cost ERP** — Budgets, expenses, milestones, utilization and profitability tracking — Port `3076`
18. **Legal Office ERP** — Matters, clients, billing, documents, hearings and compliance timeline — Port `3077`
19. **Energy Asset ERP** — Assets, maintenance, field readings, incidents, procurement and safety controls — Port `3078`
20. **Holding Executive ERP** — Multi-company executive ERP with KPIs, subsidiaries, financials and risk signals — Port `3079`

## Run all

```powershell
powershell -ExecutionPolicy Bypass -File ".\INSTALL-ALL.ps1"
powershell -ExecutionPolicy Bypass -File ".\RUN-ALL-20-PORTS.ps1"
```
