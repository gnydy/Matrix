# SportX Performance Store

Professional Next.js ecommerce template — V2.

## Commercial description
Performance sports store with athlete categories, training packs, and metric-led product cards.

## Target client
Sports shops, gym equipment sellers, activewear brands.

## UX pattern
`performance-grid` — this template is not a recolor. It uses a unique store structure, homepage composition, catalog style, and commerce positioning.

## Pages
- `/` Home
- `/products` Product listing
- `/products/[slug]` Product detail
- `/cart` Cart UI
- `/checkout` Checkout mock
- `/account` Customer account mock
- `/control-room` Commerce control room / dashboard

## Tech
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock Data
- Component-based structure
- Responsive design
- SEO metadata

## Run

npm run dev -- --port 3027
