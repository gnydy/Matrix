# MediShop Pharmacy Front

Professional Next.js ecommerce template — V2.

## Commercial description
Pharmacy frontend with health categories, safe product cards, reorder reminders, and pharmacist support UI.

## Target client
Pharmacies and health product stores.

## UX pattern
`health-shelf` — this template is not a recolor. It uses a unique store structure, homepage composition, catalog style, and commerce positioning.

## Pages
- `/` Home
- `/products` Product listing
- `/products/[slug]` Product detail
- `/cart` Cart UI
- `/checkout` Checkout mock
- `/account` Customer account mock
- `/control-room` Commerce control room / dashboard

## Tech
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock Data
- Component-based structure
- Responsive design
- SEO metadata

## Run
```powershell
npm install
npm run dev -- --port 3036
```
