# JewelVault Fine Jewelry

Professional Next.js ecommerce template — V2.

## Commercial description
Fine jewelry storefront with private vault UX, appointment CTA, certification panels, and collection storytelling.

## Target client
Jewelry houses, gold stores, luxury accessory brands.

## UX pattern
`private-vault` — this template is not a recolor. It uses a unique store structure, homepage composition, catalog style, and commerce positioning.

## Pages
- `/` Home
- `/products` Product listing
- `/products/[slug]` Product detail
- `/cart` Cart UI
- `/checkout` Checkout mock
- `/account` Customer account mock
- `/control-room` Commerce control room / dashboard

## Tech
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock Data
- Component-based structure
- Responsive design
- SEO metadata
