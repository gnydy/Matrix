# Moda Atelier Fashion

Professional Next.js ecommerce template — V2.

## Commercial description
Runway-style fashion store with lookbook strips, size assistant, editorial outfit bundles.

## Target client
Fashion boutiques and premium clothing brands.

## UX pattern
`lookbook-runway` — this template is not a recolor. It uses a unique store structure, homepage composition, catalog style, and commerce positioning.

## Pages
- `/` Home
- `/products` Product listing
- `/products/[slug]` Product detail
- `/cart` Cart UI
- `/checkout` Checkout mock
- `/account` Customer account mock
- `/control-room` Commerce control room / dashboard

## Tech
- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock Data
- Component-based structure
- Responsive design
- SEO metadata

## Run
```powershell
npm install
npm run dev -- --port 3021
```

## Build

