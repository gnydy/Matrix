# QA Checklist - Corporate Websites Batch 01 V2

- Each template has a unique layout concept, not only color/text change.
- Each template contains app/page.tsx, app/layout.tsx, data/site.ts, components, README, run/build/zip scripts.
- Each template has a fixed port from 3000 to 3019.
- Test each with `npm install` then `npm run build`.
- Check responsive behavior at 390px, 768px, 1024px, and desktop.
- Check that the template answers a specific target client and industry.
