# Restaurant Systems — Category 10 — Batch 07

This batch contains 20 independent Restaurant System frontend templates. Each template is built with Next.js, TypeScript, Tailwind CSS, reusable components, mock data, responsive UI, SEO metadata, and PowerShell scripts for run/build/zip.

## Batch goal
These are not simple restaurant brochures. Each template includes restaurant-specific operating screens: menu, online ordering mock, reservations, kitchen dashboard, delivery tracking, and control room/dashboard.

## Templates

1. **TableOps Fine Dining System** — Fine dining restaurant groups — Premium reservation-first restaurant website with dining room intelligence
2. **QuickBite Delivery Hub** — Fast food and delivery chains — High-speed ordering system for delivery-first restaurants
