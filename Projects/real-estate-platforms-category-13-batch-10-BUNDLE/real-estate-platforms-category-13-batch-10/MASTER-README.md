# Real Estate Platforms — Category 13 / Batch 10

This batch contains 20 independent Next.js real estate platform templates. Each template is frontend-only, mock-data driven, responsive, and structured for portfolio display and template marketplace sale.

## Included Interfaces
- Property listings
- Search and filters
- Property detail pages
- Agent profiles
- Mortgage calculator mock
- Viewing booking flow
- Admin / CRM dashboard mock
