# GlobalRelocate Expat Homes

Relocation property platform for expats, executives, and international families.

## Use Case
Offer relocation-ready homes with school proximity, services, and concierge viewings.

## Target Client
Relocation firms and premium rental brokers

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- Component-based structure
- Mock Data driven real estate UI

## Pages
- `/` Overview landing
- `/listings` Property listings + filters
- `/property/[slug]` Property details
- `/agents` Agent profiles
- `/calculator` Mortgage calculator mock
- `/booking` Viewing booking mock
- `/dashboard` Real estate admin dashboard mock

## Install
```powershell
npm install
```

## Run
```powershell
.\run.ps1
```

## Build
```powershell
.\build.ps1
```

## Zip
```powershell
.\zip.ps1
```

## Edit Content
Change all template text, listings, agents, stats, colors, and positioning from `data/site.ts`.

## Delivery Notes
This is a frontend template with mock data. It is not a production MLS, CRM, payment, or legal contract system until connected to a secure backend.
